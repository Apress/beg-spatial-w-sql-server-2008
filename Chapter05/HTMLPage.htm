Virtual Earth Sql Server 2008 Spatial Data Generator

This page allows you to graphically define spatial objects by clicking on a Virtual Earth control, and then generate SQL statements which will insert those objects into a SQL Server 2008 geography column.

1.) Select the type of geometry to create

2.) Click on the map to define point(s) of this geometry

3.) Well-Known Text

The WKT representation of the geometry will appear here.

You can use this WKT representation in conjunction with the STGeomFromText() method.
